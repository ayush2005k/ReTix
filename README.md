**ReTix** is a modern, responsive event ticket reselling platform inspired by top-tier booking sites like BookMyShow. Our mission is to eliminate ticket fraud by providing a marketplace exclusively for **verified resellers**.

## ‚ú® Features

### Current (Frontend UI)
* **Clean Home Page:** Modern event discovery layout with search functionality.
* **Verified Badge System:** Visual distinction between verified and pending ticket listings.
* **Seller Upload Workflow:** A specialized 3-step verification process requiring:
    1. Ticket Front Image
    2. Ticket Back Image
    3. Ticket photo with a unique handwritten verification code.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

### Future Goals (Backend Implementation)
* **Secure Authentication:** User login and signup using Spring Security and JWT.
* **Admin Verification Portal:** A dashboard for admins to approve/reject ticket listings based on uploaded proof.
* **Real-time Database:** Dynamic event loading and ticket availability using MySQL.
* **Live Hosting:** Deploying the full-stack application for public use.
